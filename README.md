# BoardSorter

自动监控下载文件夹，将教学文件按学科智能分类归档的 Windows 工具。

## 使用方法

1. 将 `boardsorter.exe` 和 `config.ini` 放在同一目录
2. 修改 `config.ini` 中的路径配置（归档根目录、监控的下载文件夹）
3. 运行：
   - `boardsorter.exe --console` - 控制台模式，查看实时日志
   - `boardsorter.exe` - 静默托盘模式，后台运行
   - `boardsorter.exe --autostart` - 设置开机自启

## 配置说明

`config.ini` 中 `科目文件夹列表` 一项可随意修改，逗号分隔即可。默认：数学, 语文, 英语, 物理, 化学, 生物。

## 分类逻辑

1. 热词匹配 → 直接归档并更新命中时间
2. 冷词匹配 → 直接归档但不续期
3. AI 轻量分析（仅文件名）→ 成功则学习为热词
4. AI 深度分析（提取正文）→ 成功则学习为热词
5. 仍无法判定 → 归档到"无法确定"

每日 12:00 自动执行热词降级/冷词清理。

## 构建

```
go build -o boardsorter.exe .
```

单文件可执行，不依赖运行时环境。
